# This app fetchs news from [NEWS API](https://newsapi.org/).




## Getting Started

Create a `.env` file with `API_KEY=<insert API key here>`. [Generate API key](https://newsapi.org/register).
Save the same in the root of the project.


### Clone the repository
```bash
git clone https://github.com/lmajhi/latest-news-app.git
```
### Install all dependencies
```bash
npm install --verbose
```

### Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
